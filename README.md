# Tango (ARM32 转 ARM64 翻译器) 项目

> **项目类型**: 逆向工程与二进制翻译系统
> **核心目标**: 重构 Tango 翻译器以支持 ARM32 应用在 ARM64-only 的 Android 10 环境中运行。
